School System

STACK USED: NODE.JS | MONGODB | JS | CSS

A system designed for a primary school using node as back end framework and vanilla js for frontend.
This project consists of a main web page with a node.js server and MongoDB database created with the intent of helping a local school by providing a secure platform for their staff and comunity, also providing with an online payment platform so that the school comunity can pay efficiently.
The system counts with a role based use system where admin, teachers and students can access the platform to perform different actions such as consult schedule or basic information.
